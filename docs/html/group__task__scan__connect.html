static uint32_t validate_bleam_adv_report ( uint8_t *  p_data_uuid,
const bool  rssi_filter_passed 
)
static

Function for validating received Bleam device adv data.

Parameters
[in]p_data_uuidPointer to the advertized UUID.
[in]rssi_filter_passedBoolean value of whether the adv report RSSI value is acceptable under RSSI lower level limit.
Return values
NRF_SUCCESSIf the scanned packet was validated.
NRF_ERROR_NOT_FOUNDIf 16-bit UUID does not match Bleam UUID.
NRF_ERROR_INVALID_PARAMIf the scanned device has too low RSSI level.
NRF_ERROR_INVALID_ADDRIf this packet was addressed to another specific Bleam Scanner node.
