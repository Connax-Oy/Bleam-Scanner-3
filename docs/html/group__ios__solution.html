Detailed Description

Solution of iOS-specific background advertising problem.

Apple devices don't explicitly advertise services in background; rather, they advertise Apple-specific info that is incomprehencible to third party BLE devices. This module is a solution to detecting Bleam service running in the background on an iOS device. For details, please refer to BLEAM Scanner 2 wiki: iOS BLEAM solution.
