Detailed Description

nRF SDK DFU and bootloader operations and handlers for Bleam Scanner.

Bleam Scanner supports bootloader in order to perform a DFU over BLE. Bleam Scanner is developed to be compatible with nRF52's BLE Secure DFU Bootloader and nRF51's BLE Secure DFU Bootloader. When developing for RuuviTag, refer to the tutorial Ruuvi Firmware: Bootloader.

◆ app_shutdown_handler()

static bool app_shutdown_handler ( nrf_pwr_mgmt_evt_t  event)
static

Handler for shutdown preparation.

During shutdown procedures, this function will be called at a 1 second interval untill the function returns true. When the function returns true, it means that the app is ready to reset to DFU mode.

Parameters
[in]eventPower manager event.
Return values
trueif shutdown is allowed by this power manager handler.
falseotherwise.
Note
So far, always returns true because there are yet no conditions for denying resetting to DFU mode.
