void blesc_set_idle_time_minutes ( uint32_t  minutes)

Function to to set wakeup time for Bleam Scanner IDLE state on request.

This function sets a future value for m_blesc_wakeup_uptime for m_blesc_uptime to reach. It is called on confirmation of Bleam request to IDLE for set amount of minutes.

Parameters
[in]minutesAmount of minutes Bleam Scanner will IDLE for.
