enum config_s_fail_status_t

Configuration Service FAIL detail status type.

Enumerator
CONFIG_S_NO_FAIL 

No Configuration Service fail detected.

CONFIG_S_FAIL_BLEAM 

Configuration fail reported by Bleam Tools device.

CONFIG_S_FAIL_DATA 

Invalid data received.

CONFIG_S_FAIL_FDS 

Error while writing configuration data to flash.

CONFIG_S_FAIL_TIMEOUT 

Bleam Tools did not respond.
