◆ bleam_send_signature_chunks()

static void bleam_send_signature_chunks ( void  )
static

Function for fragmenting signature to send to Bleam.

Function for sending signature hash over to Bleam. It takes data from signature array m_signature[] and packs it into an array. When the array is full, it calls bleam_send_write_data() to write this data.

Returns
Nothing.

static void bleam_send_write_data ( uint8_t *  p_data_array,
uint16_t  p_data_len 
)
static

Function for writing data to Bleam.

This function sends the contents of p_data_array[] of size p_data_len over to bleam_service to be sent to Bleam. Function should only be called after connection to Bleam is established and the m_bleam_send_char characteristic is discovered.

Returns
Nothing.
